# vite-plugin-router-resolve

## Usage

- run `yarn add vite-plugin-router-resolve --dev`
- add the plugin into `vite.config.js`
``` js
// vite.config.js
const { createResolvePlugin } = require('vite-plugin-router-resolve')

module.exports = {
  plugins: [createResolvePlugin(/*options*/)],
}
```
